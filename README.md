# Student Record Management System

## Description
A console-based Student Record Management System developed using C++ and Data Structures.

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- Sort Students
- Undo Delete
- Recently Added Students
- File Handling

## Data Structures Used
- Linked List
- Stack
- Queue
- Bubble Sort
- Linear Search

## Technologies Used
- C++
- File Handling
- Visual Studio Code
